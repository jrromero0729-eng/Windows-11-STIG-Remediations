# WN11-CC-000039 – Remove "Run as different user" from Context Menus

This folder contains remediation and verification scripts for DISA STIG WN11-CC-000039 on Windows 11.

## Summary
Implements DISA STIG WN11-CC-000039 (V-253336) by removing the "Run as different user"
option from Windows context menus to reduce the risk of privilege misuse.

## STIG Details
- STIG ID: WN11-CC-000039
- Vulnerability ID: V-253336
- Policy Area: Computer Configuration → Administrative Templates → MS Security Guide
- Policy Name: Remove "Run as different user" from context menus

- remediation.ps1
- verification.ps1

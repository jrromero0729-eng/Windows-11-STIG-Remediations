# WN11-AC-000005 – Account Lockout Duration

This folder contains remediation and verification scripts for DISA STIG WN11-AC-000005 on Windows 11.

## Summary
Implements DISA STIG WN11-AC-000005 (V-253297) by enforcing an account lockout duration
of 15 minutes to mitigate brute-force authentication attacks.

## STIG Details
- STIG ID: WN11-AC-000005
- Vulnerability ID: V-253297
- Required Duration: ≥ 15 minutes
- Policy Area: Account Policies → Account Lockout Policy

## Included Files
- remediation.ps1
- verification.ps1
